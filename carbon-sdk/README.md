## Installation
Carbon SDK can be installed using your preferred package manager.
### pnpm
`pnpm i @raresloth/carbon-sdk`
### npm
`npm i --save @raresloth/carbon-sdk`
### yarn
`yarn add @raresloth/carbon-sdk`
